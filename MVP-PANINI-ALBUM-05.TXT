## PARTE 2 — LISTA CORTA

La lista corta debe agrupar IDs por prefijo.

Ejemplo esperado:

Mis faltantes Panini World Cup 2026:

FWC1,8
MEX05,08,09,10,12
RSA02,07,09,17
KOR08,09,17

Reglas:

1. Para IDs normales:
   MEX05, MEX08, MEX09
   debe convertirse en:

MEX05,08,09

2. Para FWC:
   FWC1, FWC8
   debe convertirse en:

FWC1,8

3. Para 00:
   debe mostrarse como línea propia:

00

4. Ordenar por prefijo alfabéticamente.

5. Ordenar números de menor a mayor.

6. Mantener el primer número con el prefijo.
   Los siguientes números van separados por coma, sin repetir prefijo.

Ejemplo:

COL05
COL08
COL10

Resultado:

COL05,08,10

7. Si hay IDs raros que no matchean patrón, ponerlos al final como líneas independientes.

## PARTE 3 — LISTA EXTENDIDA

La lista extendida debe agrupar por país/sección/contexto.

Formato deseado:

Mis faltantes Panini World Cup 2026:

Colombia
COL05 - Nombre jugador
COL08 - Nombre jugador

México
MEX05 - Nombre jugador
MEX08 - Nombre jugador

Sudáfrica
RSA02 - Nombre jugador

SECCION 4
FWC1 - Descripción
FWC8 - Descripción

Reglas:

1. Agrupar usando preferentemente:

item.context

Si `item.context` no existe, usar:

item.section

Si tampoco existe, usar:

"Otros"

2. Dentro de cada grupo, ordenar por ID.

3. Cada línea debe tener:

ID - Nombre

Ejemplo:

COL05 - Nombre jugador

4. No repetir país/sección en cada línea si ya aparece como título del grupo.

Antes no deseado:

MEX05 - Nombre jugador - México
MEX08 - Nombre jugador - México

Ahora deseado:

México
MEX05 - Nombre jugador
MEX08 - Nombre jugador

5. Si el nombre está vacío, usar solo el ID.

Ejemplo:

COL05

6. Para FWC usar la descripción/nombre que venga desde Google Sheets.

Ejemplo:

SECCION 4
FWC9 - ITALIA 1934
FWC10 - BRASIL 1950

function buildShortMissingText() {
const groups = {};
const extras = [];

missingItems.forEach(item => {
const id = String(item.id || "").trim().toUpperCase();
if (!id) return;

```
if (id === "00") {
  extras.push("00");
  return;
}

const match = id.match(/^([A-Z]+)(\d+)$/);
if (!match) {
  extras.push(id);
  return;
}

const prefix = match[1];
const number = match[2];

if (!groups[prefix]) {
  groups[prefix] = [];
}

groups[prefix].push(number);
```

});

const lines = Object.keys(groups)
.sort()
.map(prefix => {
const numbers = groups[prefix]
.sort((a, b) => Number(a) - Number(b));

```
  if (numbers.length === 0) return "";

  return prefix + numbers[0] + (
    numbers.length > 1
      ? "," + numbers.slice(1).join(",")
      : ""
  );
})
.filter(Boolean);
```

extras.sort().forEach(x => lines.push(x));

return "Mis faltantes Panini World Cup 2026:\n\n" + lines.join("\n");

## PARTE 7 — IMPLEMENTACIÓN SUGERIDA LISTA EXTENDIDA

Implementar una función similar a:

function buildExtendedMissingText() {
const groups = {};

missingItems.forEach(item => {
const id = String(item.id || "").trim();
if (!id) return;

```
const groupName = String(
  item.context ||
  item.section ||
  "Otros"
).trim();

const name = String(item.name || "").trim();

if (!groups[groupName]) {
  groups[groupName] = [];
}

groups[groupName].push({
  id,
  name
});
```

});

const lines = ["Mis faltantes Panini World Cup 2026:", ""];

Object.keys(groups)
.sort()
.forEach(groupName => {
lines.push(groupName);

```
  groups[groupName]
    .sort((a, b) => a.id.localeCompare(b.id, "es", { numeric: true }))
    .forEach(item => {
      if (item.name) {
        lines.push(item.id + " - " + item.name);
      } else {
        lines.push(item.id);
      }
    });

  lines.push("");
});
```

return lines.join("\n").trim();
}

## PARTE 8 — FUNCIÓN COPY GENÉRICA

Crear función genérica para copiar:

async function copyText(text, buttonId, originalLabel) {
try {
await navigator.clipboard.writeText(text);
document.getElementById(buttonId).textContent = "Lista copiada";
setTimeout(() => {
document.getElementById(buttonId).textContent = originalLabel;
}, 1800);
} catch (error) {
alert(text);
}
}

Luego:

async function copyShortList() {
copyText(
buildShortMissingText(),
"copyShortButton",
"Copiar lista corta"
);
}

async function copyExtendedList() {
copyText(
buildExtendedMissingText(),
"copyExtendedButton",
"Copiar lista extendida"
);
}
